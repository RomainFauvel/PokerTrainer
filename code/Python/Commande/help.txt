Bienvenue dans Poker Trainer
________________________________________________

Dans ce logiciel, vous pourrez vous entraîner sur différents scénarios.
Pour importer un scénario, vous devez:
    -Revenir dans la fenêtre "Home"
    -Cliquer sur "Select Configuration"
    -Sélectionner votre fichier scénario
Une fois sélectionné, vous pouvez cliquer sur le bouton "Play" pour vous entraîner sur le scénario.

Remarque: Par défaut, le scénario sélectionné est un scénario aléatoire.


Amusez-vous bien ;)
