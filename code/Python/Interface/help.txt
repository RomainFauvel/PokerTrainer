Bienvenue dans Poker Trainer
________________________________________________

Dans ce logiciel, vous pourrez vous entraîner sur différents scénarios que vous aurez importés en amont.
Pour importer un scénario, vous devez:
    -revenir dans la fenêtre "Home"
    -Cliquez sur "Select Configuration"
    -Sélectionner votre fichier scénario
    -Une fois sélectionnée, vous pouvez cliquez sur le bouton "Play" pour vous entraînez sur le scénario

Remarque: Par défaut, le scénario sélectionné est un scénario basique


Amusez-vous bien ;)
